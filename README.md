# Space Battle (Arcade Project)

2D-шутер в космосе с уникальной механикой управления щитами. Игроку необходимо маневрировать, грамотно распределять энергию между щитами и уничтожать вражеские корабли.

Игра вдохновлена вселенной Warhammer 40000. В игре использован саундтрек из игры Battlefleet Gothic: Armada, а так же спрайты боевого корабля Империума и кораблей-каменюг орков.

## Как играть (Управление)

Игра управляется с клавиатуры и мыши:

### Движение и стрельба
| Клавиша | Действие |
| :--- | :--- |
| **W** | Включить двигатели (ускорение) |
| **A / D** | Поворот корабля влево/вправо |
| **ЛКМ (Левая кнопка)** | Залп из левого борта |
| **ПКМ (Правая кнопка)** | Залп из правого борта |

### Система щитов
| Клавиша | Действие |
| :--- | :--- |
| **1** | Выбрать задний щит (BACK) |
| **2** | Выбрать правый щит (RIGHT) |
| **3** | Выбрать передний щит (FRONT) |
| **4** | Выбрать левый щит (LEFT) |
| **SPACE** | Перекачать энергию в выбранный щит |

### Интерфейс
| Клавиша | Действие |
| :--- | :--- |
| **ESC** | Пауза / Выход в меню |
| **P** | Показать векторы направления (отладка) |

Внутри игры есть дополнительная справка об управлении

## Запуск приложения

Для запуска игры вам потребуется Python 3.8+ и библиотека `arcade`.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/ArseniyAngel/Space-Battle.git
    cd Space_battle
    ```

2.  **Создайте виртуальное окружение (рекомендуется):**
    ```bash
    python -m venv .venv
    .venv\Scripts\activate  # Для Windows
    # source .venv/bin/activate  # Для Linux/Mac
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Запустите игру:**
    ```bash
    python main.py
    ```

## Применённые технологии

В проекте реализованы следующие технологии из курса **Arcade**:

1.  **Архитектура и ООП:**
    *   Разделение на модули (`views`, `entities`, `utils`, `config`).
    *   Использование классов `arcade.View` для разных состояний игры (Меню, Игра, Пауза, Конец игры).
    *   Наследование от `arcade.Sprite` для создания кастомных сущностей (Игрок, Враг, Частицы).

2.  **Графика и Анимация:**
    *   **Спрайты:** Использование изображений для кораблей и пуль.
    *   **Анимация:** Вращение корабля, плавное затухание частиц (alpha-канал), эффекты взрывов.
    *   **Параллакс-эффект:** Имитация камеры через движение слоёв звёзд с разной скоростью.
    *   **Отрисовка примитивов:** Рисование щитов (эллиптические дуги) и интерфейса (полоски HP/энергии).

3.  **Физика и Логика:**
    *   **Коллизии:** Обнаружение столкновений (`check_for_collision`) между пулями, врагами и игроком.
    *   **Физический движок:** Реализация инерции, ускорения и сопротивления среды (drag) для плавного движения корабля.
    *   **Математика:** Использование тригонометрии (`sin`, `cos`, `atan2`) для расчёта векторов движения, стрельбы под углом и вращения щитов.

4.  **Система частиц:**
    *   Классы для создания эффектов взрывов (`ExplosionParticle`, `Shockwave`) и шлейфа двигателей (`EngineParticle`).

5.  **Звук:**
    *   Класс `SoundManager` для управления фоновой музыкой и звуковыми эффектами (выстрелы, попадания, взрывы).
    *   Зацикливание музыки и регулировка громкости.

6.  **Игровые механики:**
    *   **Разные уровни сложности:** Easy, Medium, Hard и Endless Mode (бесконечный режим с прогрессирующей сложностью).
    *   **Сложная механика урона:** Урон распределяется по секторам щита в зависимости от угла атаки.
